USB Serial Connection Web Serial API
Connect your ESP32 directly via USB cable.
The dashboard reads the same JSON stream as BLE — no WiFi or Bluetooth needed.
⚠ Requires Chrome or Edge on desktop. Not supported on mobile or Firefox.
Connection Steps
⚠ Important: Web Serial only works when the page is served over http://localhost or https://.
Opening file:// directly from your desktop will not work for USB Serial.
Quick fix: Run python3 -m http.server 8080 in the folder, then open http://localhost:8080/dashboard_PRO.html
  1. Plug ESP32 into your PC via USB cable
  2. Make sure the sketch is running (OLED shows calibration steps)
  3. Click Connect USB above
  4. Select your COM port from the browser popup (usually "USB Serial Device" or "CP210x")
  5. Data streams at 115200 baud · JSON on every line

USB vs BLE — When to use which?
🔌 USB Cable
  • No pairing needed
  • Zero latency
  • Reliable — no dropouts
  • Can also charge ESP32
  • Great for clinic/lab use
📡 Bluetooth (BLE)
  • Wireless — patient can move
  • Works on Android phones
  • No cable clutter
  • Battery powered use
  • Great for wearable use
